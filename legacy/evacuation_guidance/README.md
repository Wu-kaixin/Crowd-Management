# Legacy Evacuation Guidance Archive

This folder preserves the previous evacuation-oriented research line.

It contains:

- `configs/`: old evacuation and two-exit bottleneck scenario files.
- `reports/`: old Stage 1-4 reports, figures, CSV summaries, and GIF media.
- `scripts/`: original CLI implementations for baseline, guided, density-DBACT, multi-seed, Stage 4, and visualization experiments.

The active project direction is now ABCG static unknown-crowd containment. Legacy DBACT and density-aware evacuation methods are kept only as historical prototypes and comparison baselines.
